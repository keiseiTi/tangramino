# 介绍

Tangramino 是一款低代码前端框架，专为快速构建可视化页面与事件流程而生。无论是专业开发者，还是非技术背景的产品与运营人员，都能通过拖拽式视图搭建与可视化事件逻辑编排，零代码或极低代码地完成复杂页面的开发。

Tangramino 提供开箱即用的物料和完备的事件节点，覆盖常见业务场景；同时，支持自定义物料、插件扩展与编辑器二次开发，赋予你完全的控制权与灵活性。无需从零搭建底层架构，Tangramino 让你专注于业务创新——快速构建、灵活迭代、持续进化。

现在，基于 Tangramino，打造属于你的专属低代码前端平台吧！

## 特性

- 可视化搭建双引擎：视图 + 事件逻辑
  支持拖拽式页面布局搭建与可视化事件流程编排，实现从界面到交互的全链路低代码开发。无需编写模板或逻辑代码，即可完成动态交互页面的构建。

- 开箱即用的丰富生态
  内置常用 UI 物料组件、通用插件体系与完整的事件节点库，覆盖表单、列表、条件判断、数据请求等典型场景，开箱即用，快速启动项目。

- 高度可定制与可扩展
  全面支持自定义组件、插件扩展、编辑器和渲染引擎定制。无论是增强功能还是深度集成，Tangramino 均可灵活适配企业级需求，助力打造专属低代码平台。
